À propos

  • Raitoshi : jeu vidéo de combat gladiateur
  • Projet individuel (Unity / C#)

Jeu d’action où le joueur incarne un gladiateur combattant des monstres dans une arène. Fonctionnalités : déplacements, sauts, attaques à l’épée, ennemis avec points de vie, décors dynamiques et objets collectables. Évolutions : compétences débloquables, portage Web (WebGL) et sécurisation du code.

Expériences

Développement d’un jeu vidéo en C# sous Unity :

  • Système de combat et scripts modulaires
  • Portage Web (Netlify) et sécurisation (SonarQube, OWASP ZAP)
  • Expérience complète du prototype à la mise en ligne sécurisée
